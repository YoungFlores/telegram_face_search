# Telegram Face Recognition Parser

Скрипт парсит сообщения и изображения из Telegram-канала за последние 4 месяца и ищет совпадения одного и того же человека по лицу на разных изображениях, прикреплённых к одному сообщению.

---


## Установка

### 1. Получить API_ID и API_HASH

1. Зайдите на [my.telegram.org](https://my.telegram.org) и войдите через свой номер телефона
2. Нажмите **"API development tools"**
3. Заполните форму (App title и Short name — любые)
4. Нажмите **"Create application"**
5. Скопируйте `App api_id` и `App api_hash`

### 2. Создать виртуальное окружение

```bash
python -m venv venv
source venv/bin/activate       # Linux / macOS
venv\Scripts\activate          # Windows
```

### 3. Установить зависимости

```bash
pip install setuptools
pip install telethon dlib face_recognition Pillow numpy
pip install git+https://github.com/ageitgey/face_recognition_models
```

---

## Запуск

```bash
TG_API_ID=ваш_api_id TG_API_HASH=ваш_api_hash python telegram_face_search.py
```

При первом запуске скрипт запросит:
- **Номер телефона** — введите в формате `+79991234567`
- **Код подтверждения** — придёт в приложение Telegram (чат от отправителя "Telegram")
  

После авторизации создаётся файл `tg_session.session` — повторный ввод данных не потребуется.

---

## Настройки скрипта

Параметры находятся в начале файла `telegram_face_search.py`:

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `CHANNEL` | `kaliningradru` | Username канала |
| `MONTHS` | `4` | Глубина поиска в месяцах |
| `TOLERANCE` | `0.55` | Порог схожести лиц (0–1, меньше = строже) |
| `SAVE_IMAGES` | `True` | Сохранять ли скачанные фото на диск |
| `SAVE_DIR` | `downloaded_images` | Папка для сохранения фото |
